General

Element Typical Parent Description
<header> <body> The topmost visible element.
<nav> <header> The site navigation links.
<section> <body> A thematic grouping of elements
<article> <body> Independent self-contained content.
<aside> <section> <body> Indirectly related to surrounding content.
<details> <article> <section> Toggleable additional details.
<footer> <body> Footer for a document or section.
<figure> <body> <section> Self-contained content, i.e. photos, illustrations, etc.
<figcaption> <figure> Defines a caption for a photo.
<main> <body> Specifies the main content.
<mark> ANY Defines text to be marked or highlighted.
<summary> <details> Defines a visible heading for the <details> element.
<time> ANY Defines a specific time/datetime.

Text

Element Typical Parent Description
<h1> ANY Main heading element.
<h2> ANY Secondary heading element.
<h3> ANY Sub-heading element.

Multimedia

Element Typical Parent Description
<audio> ANY Embed sound in a document.
<img> ANY Embed an image in a document.
<video> ANY Embed a media player to play a video.

Table

Element Typical Parent Description
<table> ANY Wrapper for elements to display in table format.
<thead> <table> Set of rows defining column headers in a table.
<tbody> <table> The set of rows containing table data.
<tr> <tbody> Table row container.
<td> <tr> Table cell data container.
<tfoot> <tbody> Set of rows that define the table footer.
